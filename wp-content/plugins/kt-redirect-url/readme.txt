== Description ==

This plugin created for redirecting URLs to specified Link under plugin setting. This plugin is most useful for those who are using premium themes( Free themes as well) and while importing demo content theme creates Unusual posts, pages, menus and more, in such cases site owner/ developer does't want to remove them permanantly but it headache for google ranking so to avoid such "404 & unuseful posts having dummy contents" and theme data to keep safe and rank over google.

= Features =

*  Extremely easy to use

*  No burden on the database

*  one step you to go here and avoid 404

* Improve ranking over SEO

* Redirect all URL/Links which are not listed under posts/menu dropdown on "Redirect ALL" Setting page.

== Installation ==

1. Upload kt-redirect-url folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Setting "Redirection ALL" and select menus & posts which are not to be redirected as well as don't forget to set "Enable" option.
